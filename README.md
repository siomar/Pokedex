# Pokedex

Este projeto é uma aplicação móvel de Pokedex desenvolvida com React Native. A aplicação lista Pokémons, mostra detalhes sobre cada um deles e permite armazenar Pokémons selecionados pelo usuário.

## Funcionalidades

- **Listagem de Pokémons**: Visualize uma lista de todos os Pokémons disponíveis.
- **Detalhes do Pokémon**: Veja detalhes individuais de cada Pokémon, incluindo habilidades e espécies.
- **Armazenamento de Pokémons Selecionados**: Capture Pokémons.

## Requisitos

- Node.js
- Yarn ou npm
- React Native CLI
- Android Studio (para desenvolvimento Android)
- Xcode (para desenvolvimento iOS)

## Instalação de Dependências

1. Clone o repositório:

   ```sh
   git clone https://github.com/siomar/Pokedex.git
   cd Pokedex

   ```

2. Instale as dependências:

   ```sh
   yarn install
   # ou
   npm install

   # para ios
   cd ios
   pod install
   cd ..
   ```

3. Rodando o Projeto

```sh
# para android
yarn android

# para ios
yarn ios

```
